## 🎨 일관성 유지 전략

### 1. 같은 Seed 사용 (강력 추천)
```
1. Level 1 생성 후 이모지로 seed 확인
2. 모든 레벨에 --seed [번호] 추가
3. 얼굴/체형이 일관되게 유지됨
```

### 2. Character Reference 사용
```
/imagine [프롬프트] --cref [Level_1_URL] --cw 100
```
- 첫 번째 이미지를 캐릭터 레퍼런스로 사용
- `--cw 100`: 최대 일관성

### 3. Style Reference (선택)
```
/imagine [프롬프트] --sref [Level_1_URL] --sw 50
```
- 같은 촬영 스타일 유지
