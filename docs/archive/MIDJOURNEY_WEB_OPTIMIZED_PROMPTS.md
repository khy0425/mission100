```

"VS" → 대결 구도로 인식

```

"overlapping stages" → 겹치는 단계
"natural flow" → 자연스러운 흐름
```

```

"far left" → 맨 왼쪽

"center" → 중앙
"upper third" → 상단 1/3
"lower third" → 하단 1/3
```

```
